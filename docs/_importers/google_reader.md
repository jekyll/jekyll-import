To import your posts from a [Google Reader](http://reader.google.com) XML dump file, run:

{% highlight bash %}
$ ruby -rubygems -e 'require "jekyll-import";
    JekyllImport::Importers::GoogleReader.run({
      "source" => "my_file.xml"
    })'
{% endhighlight %}

The `source` field is required.
